# Tech Stack

- **Core**: React (v19.2.6), Vite (v8.0.12)
- **Routing**: React Router DOM (v7.17.0)
- **Charts**: Recharts (v3.8.1)
- **Icons**: Lucide React (v1.17.0)
- **Styling**: TailwindCSS (v3.4.19), PostCSS, Autoprefixer.
- **State Management**: Zustand (v5.0.14)
- **Network / API**: Axios (v1.17.0)
- **Real-time**: socket.io-client (v4.8.3)
- **Date manipulation**: date-fns (v4.4.0)
